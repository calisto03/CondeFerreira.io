O gráfico apresentado é um Diagrama de Sankey.

Cada linha representa um trajeto específico das viagens a mando de Joaquim Ferreira dos Santos.
Este diagrama é lido da esquerda para a direita, começando pelos locais de partida das embarcações, passando por duas passagens intermédias até aos locais de destino.

Basta passar com o cursor por cima das linhas para obter mais informações sobre a quantidade de pessoas traficadas em cada rota de tráfico.

As cinco visualizações apresentadas são Gráficos de Barras Circulares.

Cada visualização é individual e pretende mostrar as relações existentes entre dados referentes às viagens feitas para traficar pessoas escravizadas.

Se o utilizador quiser saber sobre os dados presentes na base das barras deve passar com o cursor por cima das barras.

Quando o símbolo "-" estiver presente nas visualizações, significa que não há valores em relação a esse dado específico.
